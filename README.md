# Losowanie pytań

Prosta strona z backendem w Pythonie. Pytania są losowane po stronie serwera i zwracane do frontendu przez API.

## Uruchomienie

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Po uruchomieniu wejdź na adres:

```text
http://127.0.0.1:5000
```
